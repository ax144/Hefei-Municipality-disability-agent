# 合皖助联 - 残疾人就业市场智能提醒服务

## 📌 项目简介

自动监测合肥市残疾人就业市场动态，识别商业合作机会，每2天推送真实有效的就业信息到飞书群。

## 🎯 核心功能

- ✅ **招聘信息**：自动搜索残疾人专属岗位（数据标注、手工制品、电商客服等）
- ✅ **政策信息**：抓取政府网站最新政策（就业补贴、培训扶持等）
- ✅ **招标信息**：识别甲方发包需求（排除乙方找活、服务商推荐）
- ✅ **产品信息**：辅助设备、智能硬件、专用软件新品推荐
- ✅ **日期过滤**：只推送最近30天内的有效信息
- ✅ **垃圾过滤**：排除研究报告、榜单、排行榜、服务商推荐等
- ✅ **飞书推送**：自动推送到指定飞书群

## 📅 推送频率

- **自动推送**：每2天上午9点
- **手动触发**：支持 GitHub Actions 手动运行
